# RFC4122

An implementation of [RFC4122](http://www.ietf.org/rfc/rfc4122.txt)
providing v4 and v5 Universally Unique IDentifiers (UUID) URN
namespaces. [Apache](http://www.apache.org/licenses/LICENSE-2.0) licensed.
